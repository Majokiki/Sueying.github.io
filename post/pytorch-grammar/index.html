用Pytorch实现以下的操作

Create a random tensor of shape 5x3 in the interval [3, 7)

3 + torch.rand(5, 3) * (7 - 3)
​tensor([[4.8155, 4.7534, 6.3560], [3.3288, 6.5717, 6.3192], [3.7567, 3.0715, 5.6321], [5.8696, 5.9009, 4.9031], [3.5128, 4.7134, 5.5808]]) ​

Create a tensor with values from a normal distribution (mean=0, std=1).

a = torch.rand(3, 3)
a.normal_()
​tensor([[-0.2054, -1.4342, -2.1055], [ 1.5469, -0.7705, 1.9906], [-0.8077, 0.3482, -0.8980]]) ​

Retrieve the indexes of all the nonzero elements in the tensor torch.Tensor([1, 1, 1, 0, 1]).

a = torch.Tensor([1, 1, 1, 0, 1])
torch.nonzero(a)
​tensor([[0], [1], [2], [4]]) ​

Create a random tensor of size (3,1) and then horizontally stack 4 copies together.

a = torch.rand(3, 1)
a.expand(3, 4)
​​tensor([[0.5178, 0.5178, 0.5178, 0.5178], [0.2181, 0.2181, 0.2181, 0.2181], [0.6165, 0.6165, 0.6165, 0.6165]]) ​

torch.bmm(a, b)
